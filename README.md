# SEKSOV Telegram Bot

Telegram-бот для личной фиксации введения одного препарата у нескольких пользователей. У каждого Telegram-пользователя отдельные партии раствора, история введений и остатки.

## Возможности

- создание новой партии раствора только если нет активной партии;
- фиксация введения через кнопки;
- выбор только разрешённых способов и мест введения;
- просмотр текущего статуса партии;
- просмотр последнего введения с остатком после введения;
- просмотр истории с остатком после каждого введения;
- автоматический расчёт остатка текущей партии.

## Быстрый старт

```bash
python -m venv .venv
source .venv/bin/activate
pip install -e '.[dev]'
cp .env.example .env
# заполните BOT_TOKEN в .env
seksov-bot
```

## Настройки

| Переменная | Описание | По умолчанию |
| --- | --- | --- |
| `BOT_TOKEN` | токен Telegram-бота от BotFather | обязательно |
| `DATABASE_PATH` | путь к SQLite-базе | `data/seksov.sqlite3` |
| `STANDARD_DOSE_ML` | стандартный объём одного введения | `1.0` |

## UX

Основные действия доступны с главной клавиатуры:

- `➕ Новая партия`
- `✅ Зафиксировать 1 мл` при стандартном `STANDARD_DOSE_ML=1.0`; если настройка изменена, объём в кнопке меняется автоматически
- `📊 Статус`
- `🕘 Последний приём`
- `📜 История`

После фиксации бот сразу показывает текущий приём, предыдущий приём, способ, место и остаток в текущей партии. Если остаток активной партии меньше стандартного введения, бот завершает эту партию и предлагает создать новую, чтобы пользователь не застрял с неиспользуемым остатком.
